# g.cairocomp

**Category**: general

## Description

Overlays multiple X Pixmaps.

## Detailed Description

g.cairocomp isn't meant for end users.

g.cairocomp is similar to g.pnmcomp , except that it
works with X Pixmaps instead of PNM files.
